贪官陈建暖又来选举了

View on GitHub Download .zip Download .tar.gz

最近，龙车村（福建省漳平市永福镇管辖）又将举行村长选举，为什么说“又”呢？因为在今年8月底已经举行过一次，但是当时贪官陈建暖带着一帮人大闹选举现场导致没选成。作为前任村长同时因贪污等违法行为而被取消候选人资格的陈建暖为什么要干扰选举呢？因为只有重新掌权，他以前干的修路和路灯项目贪污、骗征地补偿款等勾当才不容易被调查，才有机会销毁证据，之前和他勾结过的政府官员的才能继续包庇他，别人想将他绳之以法也就不容易了。

陈建暖这回在永福镇政府在包庇下直接成候选人了，他还在做贪官的美梦。但是，他低估了我们惩治贪官、保卫家园的决心，不管陈建暖和多少贪官勾结（据说还有什么后台撑腰），官官相互，我们也誓要和贪官斗争到底。目前我们正在和北京大学、中国人民大学的朋友联系，以获得正义力量的帮助，同时我们也正准备通过适当途径和媒体（人民日报、中央电视台等）朋友联系，希望获得社会各界人士的支持，并且在做最坏的打算——去市、省甚至北京上访。也希望大家帮忙转发，你们的支持让我们更有信心走下去。
